# K 线指标设置页完整落地计划

## Summary

- 新增一个独立的 指标设置 Activity，作为 K 线页右侧齿轮入口对应的二级页面。
- 页面固定分成两大区块：主图指标、副图指标，列表样式与 Image #2/#5/#6 一致。
- 每个指标项点击后，从底部弹出各自独立的配置面板，视觉和字段分别对齐：
    - MA -> Image #1
    - BOLL -> Image #2
    - VOL -> Image #3
    - MACD -> Image #4
    - RSI -> Image #5
    - KDJ -> Image #6
- v1 真实生效字段锁定为：勾选开关 + 参数值 + 颜色。
- 线宽、线型/样式 先完整展示与保存，但不驱动图表渲染；后续可继续接入。
- 数据不做迁移处理，因为该功能与 K 线页同步首发，默认配置直接作为初始值。

## Key Changes

### 1. 配置模型与持久化

- 新增完整的 K 线指标配置模型，不再只保存“当前主图指标 / 当前副图指标”。
- 配置根模型至少包含：
    - selectedMainIndicator
    - selectedSubIndicator
    - maSettings
    - bollSettings
    - volSettings
    - macdSettings
    - rsiSettings
    - kdjSettings
- 模型结构按图收口：
    - MA
        - 10 条 MA 配置项：MA1 ~ MA10
        - 每项包含：enabled、period、style、color
    - BOLL
        - 顶部参数：period、width
        - 三条线：UP / MB / DN
        - 每项包含：enabled、style、color
    - VOL
        - 顶部样式：bullVolumeStyle、bearVolumeStyle
        - 两条均量线：MAVOL1 / MAVOL2
        - 每项包含：enabled、period、style、color
    - MACD
        - 顶部参数：shortPeriod、longPeriod、signalPeriod
        - 折线：DIF / DEA
        - 柱体：MACD bullGrow / bullShrink / bearGrow / bearShrink
        - 折线项：enabled、style、color
        - 柱体项：style、color
    - RSI
        - 三条 RSI 配置项：RSI1 / RSI2 / RSI3
        - 每项包含：enabled、period、style、color
    - KDJ
        - 顶部参数：period、kSmoothing、dSmoothing
        - 三条线：K / D / J
        - 每项包含：enabled、style、color
- AppPreferencesRepository 扩展为读写整套指标配置。
- 默认值按参考图直接初始化：
    - MA: 5 / 10 / 20 / 30 / 60 / 120 默认启用前 6 条
    - BOLL: 20 / 2.0
    - VOL: MAVOL1=5、MAVOL2=10
    - MACD: 12 / 26 / 9
    - RSI: 6 / 14 / 24，默认仅启用 RSI1
    - KDJ: 9 / 3 / 3
- 颜色默认值按参考图近似对齐，统一收口到一个默认配置对象中，后续改色只改一处。

### 2. 指标设置页结构

- 独立 Activity 顶部为简单标题栏：返回 + 指标设置。
- 页面主体是一个纵向列表，分两组：
    - 主图指标
        - MA
        - BOLL
    - 副图指标
        - VOL
        - MACD
        - RSI
        - KDJ
- 这次不在设置页中展示 EMA，因为参考图没有给出 EMA 配置面板。
- EMA 在 K 线主页面上暂时不再作为可配置项暴露：
    - 若保留渲染能力，则走内部默认参数，不出现在设置页和主页面指标切换入口
    - 更稳妥的首发方案是先从 K 线页主图指标入口移除 EMA，只保留 MA / BOLL
- 每个列表项展示：
    - 指标简称
    - 中文副标题
    - 右侧箭头
    - 若当前为已选中的主/副图指标，可增加选中态文案或高亮

### 3. 底部配置面板交互

- 点击任一指标项，打开底部弹出配置面板，样式统一为：
    - 顶部圆角大面板
    - 面板顶部拖拽条
    - 底部固定双按钮：重置、确认
- 交互规则统一：
    - 打开弹框时读取当前已保存配置到临时态
    - 在弹框内编辑只改临时态
    - 点击 确认：保存该指标配置，并将其设为当前主图/副图已选指标
    - 点击 重置：恢复该指标默认配置，但仍留在弹框内，需用户再点 确认
    - 点击遮罩或返回：丢弃临时态，不保存
- 各弹框字段按图精确对齐：
    - MA
        - 表头：指标参数 / 参数值 / 线宽 / 颜色
        - 10 行 MA 项
        - 勾选框控制启用
        - 参数值可编辑
        - 线宽列先展示为样式选择器，保存但不驱动图表
        - 颜色列为颜色选择器
    - BOLL
        - 顶部两个输入：计算周期、带宽
        - 分隔线后展示 UP / MB / DN
    - VOL
        - 顶部说明文案
        - 做多、做空 两个柱体样式选择器
        - 分隔线后展示 MAVOL1 / MAVOL2
    - MACD
        - 顶部三个输入：短周期、长周期、移动平均周期
        - 分隔线后折线配置 DIF / DEA
        - 下半区柱体样式配置：多头(增)、多头(减)、空头(增)、空头(减)
    - RSI
        - 三行 RSI1 / RSI2 / RSI3
        - 每行可开关、参数、样式、颜色
    - KDJ
        - 顶部三个输入：计算周期、移动平均周期1、移动平均周期2
        - 分隔线后 K / D / J
- 底部面板建议实现为 ModalBottomSheet，不要用 AlertDialog，因为参考图明显是底部抽屉式交互。

### 4. 选择器与编辑控件

- 参数值编辑：
    - 数值项用只读输入框样式
    - 点击后弹出小型数字输入弹框或内嵌编辑态
    - v1 直接在面板内用数字输入框即可，不再二次弹窗
- 样式选择器：
    - 所有 线宽/样式/实心空心 等字段统一做成右侧下拉选择器
    - 先支持枚举值保存
    - 图表只消费 实心/空心 于 VOL/MACD 柱体
    - 折线的“线宽/线型”先不消费，只保存
- 颜色选择器：
    - 点击后打开简单颜色面板
    - 首发不做任意取色器，固定提供参考图同风格的预置色板
    - 颜色值直接保存为 ARGB/Compose Color 对应整型
- 勾选框：
    - 仅控制该条线是否参与渲染
    - 若某指标所有线都被关闭，仍允许保存，但图表会显示为空指标状态

### 5. K 线页与图表渲染接入

- K 线页右侧齿轮保持独立 Activity 跳转。
- K 线页主副指标入口同步调整：
    - 主图指标切换改成 MA / BOLL
    - 副图指标切换改成 VOL / MACD / RSI / KDJ
    - 选中的主/副图类型由指标设置页保存结果决定
- 图表层接入真实生效字段：
    - enabled
    - period/value
    - color
- 渲染规则：
    - MA: 仅渲染启用的 MA 线，周期按配置计算
    - BOLL: 周期和带宽按配置计算，UP / MB / DN 按开关和颜色渲染
    - VOL: 柱体颜色使用做多/做空配置；均量线按配置渲染
    - MACD: DIF / DEA 按参数和颜色渲染；柱体四种颜色按涨跌分支应用
    - RSI: 按启用状态渲染 1~3 条线
    - KDJ: 按参数和颜色渲染 K / D / J
- 不在指标设置页内刷新图表。
- 返回 K 线页后，ViewModel 重新收集偏好并直接重绘当前 candles，不重新请求 K 线接口。

### 6. 代码组织

- ui/kline 下新增一套清晰的设置页子模块：
    - 指标配置模型映射
    - 设置页 ViewModel
    - 设置页 Route / Screen
    - 底部配置面板组件
    - 通用颜色选择器 / 样式选择器组件
- 图表参数默认值统一放在一个默认配置文件里，不散落在 Route / ViewModel / Engine。
- 所有新增类保留中文类级注释，关键函数写中文注释，继续遵守你当前的注释要求。

## Test Plan

- 页面结构
    - 指标设置页固定分成主图和副图两组
    - 点击 MA / BOLL / VOL / MACD / RSI / KDJ 都会打开对应底部面板
    - 面板视觉结构与参考图一致，按钮位置和表头顺序正确
- 保存行为
    - 修改参数后不点确认，关闭面板，数据不保存
    - 点确认后，配置持久化成功
    - 点重置后恢复默认值，再点确认会写回默认配置
- 图表生效
    - 返回 K 线页后，不重新请求接口，但当前图表按新参数重绘
    - 关闭某条 MA/RSI/KDJ 线后，该线消失
    - 修改 MA/BOLL/MACD/RSI/KDJ 参数后，曲线位置变化正确
    - 修改颜色后，线条/柱体颜色变化正确
    - VOL 与 MACD 柱体颜色分支正确
- 回归
    - K 线切周期、切标的、AI 分析、下拉刷新不受影响
    - 独立 Activity 返回转场正常
    - 深色主题下颜色、输入框、分隔线可读性正常

## Assumptions

- 这次功能与 K 线页一起首发，因此不做旧数据迁移。
- EMA 本轮不进入指标设置页，也不作为外显可配置项首发；若保留内部实现，则使用默认参数，不开放 UI。
- 线宽/线型 本轮作为配置项展示和保存，但不真实驱动图表；真实生效仅包含 开关 + 参数 + 颜色。
- 颜色选择器使用预置色板，不做自由取色。
- 指标设置页保存后不主动刷新 K 线，返回页面后由页面已有状态收集与重绘机制生效。
