# Changelog

## v1.0.4 (2026-04-03)

- 首页观察列表支持拖动排序与置顶 / 取消置顶
- K 线页新增 AI 对话功能，支持流式响应与会话历史
- K 线页新增搜索入口，可从图表内快速切换币对
- 统一顶部栏样式，新增涨跌弱化配色方案
- 首页长按弹出快捷菜单，支持删除、置顶、加入 / 移出悬浮窗
- 发布流程更新：等待 GitHub Actions 打包成功后再合入 main

## v1.0.3 (2026-03-31)

- 新增 K 线页面，支持多来源切换、周期切换
- 新增指标设置页，支持 MA / EMA / BOLL / VOL / MACD / RSI / KDJ 配置
- Vendored TradingView Lightweight Charts wrapper，升级 JS core 到 v5.1.0
- 图表切换单 chart + pane 结构，主图与副图共用时间轴
- 优化悬浮窗 ticker 动画与图标稳定性
- 优化首页价格更新性能，避免高频 WSS 导致滚动抖动
- 新增网络日志查看器，加强凭证存储安全性
- 绑定产物名称到应用版本号

## v1.0.2 (2026-03-28)

- 行情刷新切到 WSS 优先策略（Binance Spot / Binance Alpha / OKX Spot / OKX On-chain）
- 新增链上代币搜索与价格跟踪（OKX DEX Market API）
- 引入全局刷新协调器 + 可替换刷新引擎架构
- 行情数据走内存 StateFlow，不再每次报价写回 Room

## v1.0.1 (2026-03-28)

- 优化悬浮窗交互体验
- 修复悬浮窗快捷操作的权限与设置联动

## v1.0.0 (2026-03-27)

- 初始开源版本
- 观察列表：搜索、添加、删除 Binance / OKX 现货交易对
- 系统悬浮窗：前台服务保活、锁定拖动、透明度调节、吸附靠边
- 设置页：刷新间隔、主题、悬浮窗配置
- GitHub Actions CI/CD：自动测试、lint、签名发布
