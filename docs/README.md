# GitHub Pages 发布说明

这个目录就是 `CoinMonitor` 的 GitHub Pages 站点根目录。

## 一次性开启

1. 打开仓库设置页：`Settings -> Pages`
2. 在 `Build and deployment` 中选择 `Deploy from a branch`
3. Branch 选择 `main`
4. Folder 选择 `/docs`
5. 保存后等待 GitHub 构建完成

## 访问地址

开启成功后，项目页地址通常为：

`https://baiyanwu.github.io/CoinMonitor/`

## 当前站点资源

- `index.html`：官网结构
- `styles.css`：样式与动效
- `assets/`：图标与 4 张裁切后的应用截图

## 更新方式

后续只要修改 `docs/` 下内容并推送到 `main`，GitHub Pages 就会自动重新发布。
